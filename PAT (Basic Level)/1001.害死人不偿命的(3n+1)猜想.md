# 【PAT B-1001】害死人不偿命的(3n+1)猜想

## 题意概述
输入一个正整数n，若n为奇数，则n=(3*n+1)/2,若为偶数，则n=n/2，直到n=1为止，求循环次数。

## 输入输出格式
输入n，输出step(循环次数)。

## 数据规模
maxn=1000。

## 算法设计
while循环，用step来计数。

## C++代码
```cpp
#include<cstdio>
int main(){
    int n;
    scanf("%d",&n);
    int step=0;
    while(n!=1){
        if(n%2==1)
            n=(3*n+1)/2;
        else
            n=n/2;
        step++;
    }
    printf("%d",step);
    return 0;
}
```
